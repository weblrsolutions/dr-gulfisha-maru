# Dr. Gulfisha Maru – Personal Portfolio

Production-ready, SEO-optimized portfolio for **Dr. Gulfisha Maru** — IACH Certified Classical Homeopath and Founder of [Aayat Homoeopathic Clinic](https://aayat.clinic), Ahmedabad.

## Repository & deployment

- **GitHub:** [weblrsolutions/dr-gulfisha-maru](https://github.com/weblrsolutions/dr-gulfisha-maru)
- **Live:** https://gulfisha.com/

## Contents

- Single-page site: About, Education, Experience, Certifications, Achievements, Values, Testimonials, Book Consultation
- All booking CTAs link to **https://aayat.clinic/about.html**
- Semantic HTML5, meta tags, Open Graph, Twitter Card, JSON-LD, `sitemap.xml`, `robots.txt`

## Run locally

Open `index.html` in a browser, or use a static server:

```bash
# Python
python -m http.server 8000

# Node (npx)
npx serve .
```

## Deploy to GitHub Pages

1. Push this repo to `https://github.com/weblrsolutions/dr-gulfisha-maru.git`
2. **Settings → Pages → Source:** Deploy from branch `main` (or `master`), folder `/ (root)`
3. Save. With custom domain **gulfisha.com** set in Pages settings, the site is at https://gulfisha.com/
